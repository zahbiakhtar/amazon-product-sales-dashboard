# amazon-product-sales-dashboard

Developed an interactive Power BI dashboard to analyze Amazon product sales performance using KPI-driven analytics. Visualized YTD/QTD sales, product trends, customer reviews, and category insights with dynamic charts, DAX measures, and interactive filters for business decision-making.

# Project Overview

This project focuses on analyzing Amazon product sales data to uncover business insights related to revenue performance, customer engagement, and product trends. The dashboard was designed to help stakeholders monitor sales KPIs, identify top-performing products, and analyze category-wise sales performance using interactive visualizations.

---

# Business Problem

Businesses selling products online require efficient monitoring of sales performance and customer engagement metrics. This dashboard helps track sales growth, product demand, and review trends to support data-driven decision-making.

---

# KPIs Implemented

- YTD Sales
- QTD Sales
- YTD Products Sold
- YTD Reviews

---

# Dashboard Features

## Sales Analysis
- Sales by Month (Line Chart)
- Sales by Week (Column Chart)

## Product Performance
- Top 5 Products by YTD Sales
- Top 5 Products by YTD Reviews

## Category Insights
- Sales by Product Category Heatmap

## Interactive Filters
- Product Category Filter
- Quarter Filter

# Key Insights

- Men Shoes category generated the highest YTD sales contribution.
- Sales performance increased significantly during the final quarter of the year.
- Nikon and Atomos products ranked among the top revenue-generating products.
- SanDisk products received the highest customer review counts.
- Weekly sales trends revealed strong seasonal purchasing behavior.

# Dataset Information

- Source: Amazon Product Sales Dataset
- File Format: Excel (.xlsx)
- Data Includes:
  - Product Categories
  - Sales Amount
  - Reviews
  - Weekly & Monthly Sales Data

# How to Use

1. Download the `.pbix` file
2. Open using Power BI Desktop
3. Refresh the dataset if required
4. Explore dashboard insights using filters and slicers
